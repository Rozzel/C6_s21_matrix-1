Создает новую матрицу с заданными размерами.

Функция выделяет память для новой матрицы и инициализирует ее структуру.

Parameters
rowsКоличество строк в новой матрице.
columnsКоличество столбцов в новой матрице.
[out]resultУказатель на структуру, в которой будет храниться новая матрица.
Returns
Возвращает OK, если матрица успешно создана. Возвращает ERROR_INVALID_MATRIX, если входные параметры некорректны или память не может быть выделена. Возвращает ERROR_COMPUTATION в случае других ошибок.

Формула для выделения памяти: rows×sizeof(double *)+rows×columns×sizeof(double)
