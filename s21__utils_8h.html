Вычисляет "кофакторную" подматрицу для заданного элемента исходной матрицы.

Функция принимает квадратную матрицу ( A ), номер строки ( \text{row} ), номер столбца ( \text{col} ), и размер ( n ) матрицы. Функция вычисляет подматрицу, удаляя указанную строку и столбец, и сохраняет результат в матрице ( \text{dest} ).

int check_matrix ( matrix_t *  A)

Проверяет корректность матрицы.

Функция принимает указатель на матрицу типа matrix_t и проверяет её на корректность. Корректной считается матрица, которая не является NULL, имеет положительные размеры и содержит ненулевой указатель на двумерный массив.

Parameters
AУказатель на проверяемую матрицу.
Returns
Возвращает true, если матрица корректна; иначе false.
