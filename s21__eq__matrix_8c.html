Сравнивает две матрицы на равенство.

Функция сравнивает две матрицы на равенство, учитывая погрешность. Для сравнения элементов матриц используется значение погрешности EPSILON 1e-7.

Parameters
AУказатель на первую матрицу для сравнения.
BУказатель на вторую матрицу для сравнения.
Returns
Возвращает SUCCESS, если матрицы равны; иначе FAILURE.
